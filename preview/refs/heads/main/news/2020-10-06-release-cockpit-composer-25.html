Release of cockpit-composer 25

We are happy to announce the release of cockpit-composer 25. This release has no major features but contains useful fixes.

Below you can find the official change log. Everyone is encouraged to upgrade!

  • Improve error state messages

  • Add additional console error logging

  • Update CDP testing library

  • Update translations from weblate

  • Update minor NPM dependencies
